    JVM相关参数

    记录一些JVM参数，以供参考

    • - : 标准参数
    • -X : 非标准参数
    • -XX :
    参数 说明
    -XX:+/-UseTLAB 给对象分配内存空间时是否优先使用线程本地分配缓冲以解决在堆上分配时的竞争
    -XX:+PrintFlagsFinal 打印JVM支持的参数
    -XX:SurvivorRatio=8 新生代Eden区占80%
